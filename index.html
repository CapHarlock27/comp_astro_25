About Daneel

Daneel is a practical Python tool designed for detecting and characterizing exoplanets. Named after the robot from Isaac Asimov's science fiction, this package provides algorithms and tools for astronomical data analysis in the field of exoplanetary science.

Quick Start

Install Daneel from the GitHub repository:

git clone https://github.com/tiziano1590/comp_astro_25.git cd comp_astro_25 pip install .

Run exoplanet detection and atmospheric characterization:

# Run exoplanet detection daneel -i parameters.yaml -d # Run atmospheric characterization daneel -i parameters.yaml -a # Run both analyses daneel -i parameters.yaml -d -a

Requirements

  • Python >= 3.10
  • PyYAML for parameter file parsing
  • Additional scientific computing libraries as needed
